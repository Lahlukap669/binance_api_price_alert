# binance_api_price_alert
An program that tracks your fav crypto and sends you notification on the email if the prices rocket in either way


## Getting started
1. Download the project
2. pip install python-binance
3. run the program

## Prerequisites
* Python 3.9.2

## Built with
* python-binance
* smtp
